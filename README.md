# Project Green — 3D Viva Rehearsal Planner

Interactive Three.js rehearsal scene for the Project Green viva presentation.

## Goals
- Visualize the room layout (projector wall, posters, panel seating)
- Map five presenters to slide ranges and poster focus
- Support a step-through / timelapse of the full presentation flow

## Presenters (draft)
1. Sithum Nimhan — opening, problem, objectives, scope
2. Nadeera Prabhash — methodology, WBS, Gantt, CPM
3. Rashandi Tharushika — budget, EVM, variance, recovery
4. Yasali Sarajika — quality, risks, issues
5. Ronith Rashmikara — success criteria, lessons, launch video, closing

## Room zones (from sketch)
- Projector wall (rear)
- Plan Wall poster (left)
- Control Room poster (right)
- Viva panel table (front)
- Laptop control table
- Whiteboard
- Waiting zones for non-active speakers

## Stack
- Vanilla HTML + CSS + ES modules
- Three.js (jsDelivr CDN)
