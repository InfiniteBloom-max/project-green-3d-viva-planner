PROJECT GREEN — 3D VIVA REHEARSAL PLANNER (UPDATED)

WHAT'S NEW
- All presentation slides are included as local images extracted from your deck PDF.
- The two posters are placed in the 3D room as real poster visuals.
- All five presenters are included, color-coded, with floating name tags.
- A presentation timelapse is built in.
- Each timeline step changes:
  1) the active presenter
  2) the visible slide on the projector wall
  3) the presenter position
  4) the highlighted poster focus

FILES
- index.html : main interactive 3D rehearsal planner
- slides/ : extracted slide images from your deck PDF
- plan_poster*.png : Plan Wall poster visual
- control_poster*.png : Control Room poster visual
- sketch*.png : original room sketch (if available)
- start_server.bat : easy local server launcher
- README.txt : instructions

HOW TO RUN
1. Extract the ZIP.
2. Double-click start_server.bat.
3. Open http://localhost:8080 in Chrome or Edge.
4. Use “Play full timelapse” to watch the whole presentation sequence.

NOTES
- Three.js loads from jsDelivr, so internet access is needed.
- The system is now structured for future additions like:
  - animated poster reveals
  - camera path exports
  - custom launch video texture
  - voiceover cue markers
